柔軟なBoxLayout¶

(Tested on CPython3.9.7 + Kivy2.1.0)

公式の Box Layout よりも子の並べ方が豊富なBoxLayout。 公式の物は左→右と上→下の二種類の並べ方しかないが、これはそれらに加えて右→左と下→上にも対応する。
